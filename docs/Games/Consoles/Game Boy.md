## Quick Specs
- Manufacturer: Nintendo
- First release: April 21, 1989 (Japan); July 31, 1989 (US)
- Designed by: Nintendo R&D1 (Gunpei Yokoi, Satoru Okada)
- CPU: Sharp LR35902 (4.19 MHz)
- Memory: 8 KB RAM, 8 KB VRAM
- Display: 160x144 monochrome LCD
- Game format: Cartridges
