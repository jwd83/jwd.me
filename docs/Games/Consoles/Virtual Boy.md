## Quick Specs
- Manufacturer: Nintendo
- First release: July 21, 1995 (Japan); August 14, 1995 (US)
- Designed by: Nintendo R&D1 (led by Gunpei Yokoi)
- CPU: NEC V810 (20 MHz)
- Memory: 1 MB DRAM (system)
- Display: Dual red monochrome LED arrays (stereoscopic effect)
- Game format: Cartridges
