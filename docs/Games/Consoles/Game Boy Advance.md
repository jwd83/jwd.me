## Quick Specs
- Manufacturer: Nintendo
- First release: March 21, 2001 (Japan); June 11, 2001 (US)
- Designed by: Nintendo handheld hardware team (led by Satoru Okada)
- CPU: ARM7TDMI (16.78 MHz)
- Memory: 256 KB external WRAM + 32 KB internal WRAM + 96 KB VRAM
- Display: 240x160 TFT LCD
- Game format: Cartridges
