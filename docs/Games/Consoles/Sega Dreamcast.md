## Quick Specs
- Manufacturer: Sega
- First release: November 27, 1998 (Japan); September 9, 1999 (US)
- Designed by: Sega hardware team (headed by Hideki Sato)
- CPU: Hitachi SH-4 (200 MHz)
- Memory: 16 MB main RAM, 8 MB VRAM, 2 MB audio RAM
- Graphics: PowerVR2 (NEC/VideoLogic)
- Game format: GD-ROM (plus VMU memory cards)
