## Quick Specs
- Manufacturer: Atari, Inc.
- First release: September 11, 1977 (US)
- Designed by: Atari console team (Joe Decuir, Jay Miner, Ron Milner, Steve Mayer, Larry Wagner)
- CPU: MOS Technology 6507 (~1.19 MHz)
- Memory: 128 bytes RAM
- Graphics: Atari TIA (Television Interface Adapter)
- Game format: ROM cartridges
